Netiquette Chrome Extension - User Guide

Introduction

Welcome to the Netiquette Chrome Extension! Designed to enhance your browsing experience, Netiquette offers a suite of features tailored to make your time on the web both productive and safe. Dive into this guide to discover all that Netiquette has in store for you.

Purpose of Netiquette

Netiquette is your personal browser assistant, aiming to enhance the visual appeal of your browser, provide quick access to essential browser functionalities via shortcuts, and ensure your online safety with pro tips.

Key Features

  1. Color Theme Customization: Set your browser's mood using our intuitive color picker.
  2. Save Specific Preferences: Customize your browser functionalities for the next session.
  3. Quick Safety Tips: Get instant safety guidelines for a secure browsing experience.
  4. Keyboard Shortcuts:
    Action Shortcut
    Open New Tab Cmd + T
    Open New Window Cmd + N
    Close Tab Cmd + W
    Close Window Ctrl + Y
    Bookmark Current Page Cmd + D
